# Grand Theater Website

A beautiful, professional website for a theater showcasing performances, venues, and booking information.

## Features

- **Responsive Design** - Works perfectly on desktop, tablet, and mobile devices
- **Professional Styling** - Modern UI with elegant purple and gold color scheme
- **Multiple Pages**:
  - Home page with featured shows
  - Shows listing with detailed information
  - About page with theater history and mission
  - Contact page with form and venue information
- **Interactive Elements**:
  - Newsletter subscription
  - Contact form
  - Smooth animations and hover effects
  - Mobile-friendly navigation

## Pages

### Home (index.html)
- Hero section
- Featured shows showcase
- Quick venue information
- Newsletter signup

### Shows (shows.html)
- Complete listing of performances
- Show details, dates, and times
- Ticket booking buttons

### About (about.html)
- Theater history
- Mission statement
- Venue features
- Ways to get involved

### Contact (contact.html)
- Contact form
- Venue address and hours
- Phone and email information
- Social media links

## Styling

The website uses a professional color scheme:
- Primary: Deep Purple (#8b2252)
- Secondary: Gold (#d4af37)
- Dark Background: #1a1a1a
- Light Text: #f5f5f5

## Installation

Simply clone or download this repository and open `index.html` in your browser. No dependencies or build process required!

## Deployment

### Deploy to Vercel

1. Push this repository to GitHub
2. Go to [vercel.com](https://vercel.com)
3. Click "New Project"
4. Import your GitHub repository
5. Click "Deploy"

Vercel will automatically deploy your site and provide a live URL.

### Deploy to Other Platforms

This static website can be deployed to:
- GitHub Pages
- Netlify
- AWS S3
- Any static hosting service

## Customization
